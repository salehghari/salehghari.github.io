Weather announcement app

In this project, I wanted to practice with the API. It occurred to me to use a free and fairly functional API.

This app works in such a way that by entering the titles of the province and city, it automatically gets the weather information of that province or city from the developers.parsijoo.ir website and announces it in the shortest possible time.

Tools :

  • 'Parsijoo' website for extracting api data
  • Using javascript to insert data
  • Bootstrap for styling
